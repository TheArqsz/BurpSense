# Generate and Set API Token

**In Burp Suite:**
1. Go to **BurpSense Bridge Settings** tab
2. Click **Generate New Key**
3. Enter a name (e.g. "VS Code")
4. **Copy the token** - you won't see it again!

**In VS Code:**
1. Click the button below or use Command Palette
2. Paste the token when prompted
3. Connection will be verified automatically

The token authenticates VS Code with the bridge server.
